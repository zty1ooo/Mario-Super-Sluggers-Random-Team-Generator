# Mario-Super-Sluggers-Random-Team-Generator
Updated @EliFreedman Mario Super Sluggers Randomizer: Added Stadiums, Fixed Captain Logic, and now works on Windows 11

1. Install Python 3 from [python.org](https://www.python.org/downloads/windows/) and keep **"py launcher"** enabled during install.
2. Put all files in the same folder.
3. Edit Sluggers.bat to have your path to wherever you have the folder.
4. Double-click `Sluggers.bat`.
